# GATBL-Learning

The implementation of the paper "Air Pollutant Diffusion Trend Prediction Based on Deep Learning for Targeted Season——North China as an Example"

It is important to note that due to **confidentiality agreements**, we are unable to provide the complete dataset. We have only shared a small portion of the data, and to ensure the code runs smoothly, **we have made modifications to the model input and some of the model structure.** As a result, **the final results may differ from those presented in the paper**, and we apologize for any inconvenience this may cause.
